Text - Bild - Modul für REDAXO 4.5+
===================================

Redaxo Modul für die Pflege von Inhalten die auf einer responsiven Webseite ausgegeben werden.


Aktuelle Vorraussetzungen
-------------------------

* jQuery UI in Backend
* mForm
* Textile
* RexMarkitup

ButtonSet für RexMarkitup
